Model Architecture

Multi-view neural ranker: encode source, facts, context, then fuse and score candidates

Model addendum

Embeddings

Code tokens, fact types, path nodes, task context, and optional agent summaries each get a separate view encoder.

Fusion

Cross-view attention or gated fusion builds one candidate embedding per file/function/line window.

Heads

Rank score, rule family, affected object, evidence selection, test guidance, and UNKNOWN/confidence.

Loss functions

  • Rank: pairwise margin or listwise softmax within each task.
  • Rule/object: cross-entropy on validated labels.
  • Evidence: binary cross-entropy over linked fact IDs.
  • UNKNOWN: calibration/abstention loss for incomplete proof.
